## 🚀 Quick Start
```shell
# パッケージのインストール
yarn

# 開発サーバーの起動
yarn dev
```

## 📝 説明
alpinejsをtypescriptで書けるようにしたテンプレートです。
構成は↓の通り
- パッケージマネージャ: yarn v3
- ビルドツール: vite
